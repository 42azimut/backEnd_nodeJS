# 원격 저장소 깃허브에 실수로 올린 파일 또는 폴더 삭제 하기

## 원격 저장소와 로컬 저장소에 있는 파일을 삭제한다.
git rm 파일명

## 원격 저장소에 있는 파일 또는 폴더 삭제
git rm --cached 파일명
git rm --cached -r 폴더명

위에 작업 하고 나서 
commit 후 push 해주면 된다..
